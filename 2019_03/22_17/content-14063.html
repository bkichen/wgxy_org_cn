一、 学生会机构包括：主席、副主席、秘书处、学习部、文体部、社会实践部、女生部、生活保卫部、社团部、宣传部、广播站、心理部、国旗班（其中各部设部长一名、副部长一名、干事若干）。

二、 学生会实行主席团责任制。全主席团由主席、副主席、秘书长组成，讨论决策学生会的工作方向和重大问题，由主席同意后，呈报院团委指导老师批准后，方能实施。

三、 学生会主席团与各部门职责。

主席职责

1. 对学生会内部工作总体统筹。

2. 参加学生会例会，并组织学生会各成员的工作及活动。

3. 每周定期召开学生会例会并主持例会，指导协调学生各部工作，并及时发现问题可与辅导员沟通后提出相应的解决方案。

4. 督促检查各部的工作情况，定期检查各部的工作计划和工作总结，使其在计划和总结中，发现问题，找出缺点，不断提高工作层次和水平。及时督促各部工作的完成，帮助协调和解决遇到的问题和困难。

5. 及时把广大同学的呼声和要求向上反映，同时把上级组织的支持，辅导和帮助向下传达。

6. 直接管辖各部的日常活动并进行指导与督促。

副主席职责

1. 协助学生会主席开展工作，指定全局性工作计划并执行主席分配的任务。

2. 协助办公室进行学生会组织纪律维护，协调主席团与所辖各部门，传达主席团决定，指导所辖部门工作。

3. 对其指导、监督、检查并及时向主席汇报。

秘书处

1. 按照学生会的工作动态起草学生会规章制度的原始或修订草案，解释学生会的各项规章制度。

2. 负责学院各项活动相关信息的通知传达工作以及学生会内部信息的传达。

3. 负责文件资料、学生会各项档案的整理，各类通讯录等文件的制作。

4. 学院各类大大型活动的签到工作和节目单、邀请函的制作。

5. 负责学生会工作报告的整理、制定以及各类会议的记录工作，并列席参加学生会各个部门的例会并做相关会议记录。

6. 依照有关规定管理学生会的财务、统一管理学生会固定资产。

7. 出台学生会学期计划和总结。

8. 负责例会的组织活动。

9. 负责学生会每月工作总结并出台学生会年底总结。

10. 协助主席、副主席协调好各部门工作，监督、跟踪了解学生会各部门的工作，各项活动进展落实情况。

学习部

1. 为了促进学院学生之间的学习与文化交流，开阔学生的只是视野，组织学生参加学院及学校的文化活动，如组织或协助学院开展各种讲座等。

2. 为了促进同学学习，培养同学的各种能力为目的，及时组织开展各种以加强同学专业技能为目的的活动，如翻译大赛、演讲比赛等。

3. 加强与院领导、任课老师的联系，沟通教与学，及时反映同学对教学改革的意见和要求，配合学院的教学改革工作，以提高教学质量。

4. 与系部学生会、各班级开展有关学术反面的活动，如英语角、管理学院演讲与辩论赛，组织各种类型的演辩活动（新生辩论会、趣味辩论等），直接为大型演讲赛事培养和输送人才。

5. 对各班上的晚自习出席率的考查统计，并及时将未经请假而擅自逃课缺席的同学名单汇总上交。

文体部

1. 满足同学们的文化需求，丰富同学们的业余文化生活。积极开展有益同学身心健康、积极向上的文娱活动，如配音、表演大赛等。

2. 组织举办大型晚会等活动，如迎新晚会、运动会等。

3. 负责组织选拔文艺性人才，参加学校及各学院承办的文艺性活动及比赛，如学校大合唱、朗诵比赛等。

4. 培养院文艺骨干，及时挖掘并接纳有文艺特长的新成员，扩充本院学生文艺队伍。

社会实践部

1. 根据需要为学生会组织的大型活动对外联系经费。

2. 加强学生会与社会间沟通，开展各项公关活动，树立学生会的良好形象，同时扩大学院在全校以及社会上的影响。

3. 沟通校际，院系之间的联系，加强与兄弟院系的交流，为树立本源形象做应有的贡献。促进学院与外界进行学术、文化、艺术、文艺等反面交流。

女生部

1. 负责女生的日常生活的工作，针对女生的特点开展各项有利于女生的活动。

2. 负责组织全院女生开展女性特色的各项文艺、体育活动，反映广大女生权益，真正为女生排忧解难。

3. 负责院礼仪队的选拔和培训。

4. 负责定期召开女生代表座谈会，收集女生的意见及建议，向学校有关部门反映情况，为女生解决生活、学习困难。

生活保卫部

1. 负责我学生会活动的后勤工作。

2. 指定宿舍管理奖罚条例，宿舍评分细则及评比员管理细则。

3. 组建内务卫生各检查小组，每周对本系宿舍进行抽检并及时公布。

4. 期末评出“文明宿舍”、“优秀集体”，每周评出“优秀个人”和“最差个人”。

5. 及时向系领导反馈本系学生的意见及建议。

6. 加强配合学生会及各系学生生活交流。

社团部

1. 做好学院社团的成立，纳新，管理，并在平时社团运营中，为其提供帮助和指导，使学院社团健康向上发展。

2. 根据具体情况，以多种形式，加强社团之间的联系，加强不同社团之间的相互关系，相互学习，相互长进，共同发展。

3. 加强我院与其他兄弟学院之间的联系，同时协助其他部门进行活动。

4. 认真按时指定本部工作计划，并定期向主席团汇报工作，及时配合完成校学生会工作。

宣传部

1. 通过各种宣传渠道，传递学生会工作信息。

2. 负责学生会主办的各类活动的宣传工作和其他设计，负责学院重要事件的对外发布。

3. 培养实干的宣传干部队伍，充实和壮大宣传队伍，同时注意培养学生的宣传能力。

 

广播站

1. 广播站开播时间为每周一至周五早上6：:3-7:30，中午12:00-30，下午17：30-18：30。主持人必须在开播前10分钟到位做好播音准备。

2. 必须准时开播，不要迟到；播音时间统筹安排，不得超时。

3. 每次播音后，站长做好材料存档工作。

4. 广播内容要衔接紧密，内容健康积极；具有科学性，文化性，娱乐性。、

5. 播音时要说普通话，吐字清晰，语言流利，少出错误。

6. 广播员如有缺席，必须提前通知站长，站长做好替换准备。

7. 爱护广播器材，广播结束后要把话筒收好，站长负责督促。

心理部
1 ，心理健康部服务于全体同学，并向广大学生普及推广心理健康知识，并深入到广大同学中去，了解同学的心理健康状况，使有需要帮助的同学及时得到帮助、危机得到化解。
2，协助老师开展心理委员培训，组织学生上心理讲座课。 与心理委员交流心理健康知识并引导他们在各班开展普及，且保持联系。
3，主要负责心理健康月开展大型的心理健康教育等系列活动。
4， 增强广大同学的心理保健意识，提高全院同学的心理素质和自我调适能力，推动学校心理健康教育工作的发展。
5，协助学生会各个部门的相关工作，各成员在实践中深入学习，了解以及交流和总结。做到真正把心理健康知识融入和应用到生活中去，并做好表率。

国旗班
1.武汉工程职业技术学院国旗班于2013年11月17日由女生部自行组织成立。隶属于校团委。
2.具体负责学校每周一及例行重大节日活动的升旗仪式。
3.以提高当代大学生民族自尊心和自豪感为己任，始终发扬“爱国至上，爱校如家”的光荣传统。
4.国旗班宗旨
 自强，团结，互助。
5.誓词
①热爱国旗班集体，崇尚荣誉。
②刻苦训练，严格要求，严格遵守国旗班规章制度。
③时刻牢记国旗班宗旨和价值观。
④发扬爱国、爱家、爱校的思想。 
6.日常训练
   每周三天训练，分别是一、三、五早上六点半至七点半，如有重要活动任务将会依据情况进行加训。训练内容包括口令练习、单人及队列动作。及时提出不足与改进方案，保证每次训练都有所收获。
7.班内一切人员平等，发现问题当面解决，班长训练有误时当面指出，以便及时纠正。

四、总则

1. 学生会干部要团结一致，工作上相互支持，相互监督，要注意自身修养和形。

象。

2. 部门或干部及干事在开展工作时，要坚持工作原则，从学生会的大局出发，服从组织安排。

3. 部门或干部及干事之间若有建设性意见或不同意见，要及时向主席团反映。

4. 学生会干部应该按照“公平、公正、公开”的原则办事。若办事显失公平或者出现严重的失误，要公开检讨，并给与内部通报批评，以维护学生会的声誉。

5. 各部门开展活动前应做好详细的工作计划，及时向分管的主席汇报，征得其同意后方可进行；若未经同意，擅自开展活动，主席团将不予以支持，并给予组织者相应的处分，活动期间出现的问题由相应的组织者负全责。活动结束后，要做好总结以及收尾工作。每次活动结束一周内，活动组织者必须将活动的相关资料上交至办公留档备案。（详见档案管理制度）

6. 各部门应定期召开部门会议，由部长主持。会议须及时传达主席团形成的决议和有关文件，由于传达未到位和工作不力造成的工作损失，由各部长负责。学生会各部门开展的工作由主席团加强监督。

7. 各部门应该加强与学校相关部门以及院系之间的联系建立网络以便整个学生会工作的开展。

8. 学生会进行的不定期抽查或者检查活动，在活动没有结束之前，一律不准向外部通风报信，内部会议尚未形成的决议不得外泄。若泄露，经查实由此造成的一切后果由其负责，并视情节严重做出相应的处分。

9. 学生会干部应实事求是，忠于职守，严禁利用职务之便为本人、少数同学、本系、本班同学谋利益，严禁一个人名义代表学院向外界做出各种承诺。否则经调查核实，视情节严重，给予相应处理。

10.爱护学生会的财务。若有丢失须由当事人负责赔偿。若外借，经办公室统一，并做好记录。

11.办公室值班按原值班制度进行，值班人员安排按实际情况而定。

（注：学生会干部应该自觉遵守学生会的各项规章制度，努力不断完善个人综合素质；若出现违纪行为，视情节严重做出相应处理。1、口头警告2、书面检讨并给予内部通报批评）
